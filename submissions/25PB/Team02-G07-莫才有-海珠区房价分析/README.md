## 决策主体与选题摘要

本报告面向**有意在广州海珠区置业的潜在购房者**（主要读者）和**关注广州楼市的财富管理顾问**（次要读者）。

核心问题：海珠区房价从 2024 年高点已回落多少？当前月供压力比高峰期改善了多少？成交量是否可以作为判断市场拐点的先行指标？本报告基于 2016—2026 年的房价、利率和成交量数据，通过 5 张核心图表系统回答上述问题。

## 数据来源

- 海珠区月度二手房均价：链家/安居客/58同城（经 gotohui.com 聚合），2016.01—2026.05
- 广州全市二手房均价：安居客/58同城，2016—2026（年度）
- 5年期 LPR：中国人民银行（经 akshare Python 库获取）
- 分板块均价：58同城，2026.05
- 成交量估算：广州住建局月度网签报告 + 乐有家季度报告
- 政策事件：政府官网/新闻，手动整理

> 所有数据均为公开数据，不含付费数据库（Wind、CSMAR）内容。
